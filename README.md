# 🗂️ Figure Database Web App

A web application for organizing, sharing, and discussing anime figure collections. Users can search figures, manage personal lists, upload real-life images, and connect with other collectors.

## 🚀 Features

- 🔍 **Figure Search**  
  Search figures by name, series, maker, scale, and more.

- 🗃️ **User Collections**  
  Create and manage personalized lists:  
  - Plan to Buy  
  - Owned  
  - Wishlist  

- 💬 **Comment Section**  
  Share opinions or ask questions on each figure's page.

- 🖼️ **User Image Uploads**  
  View and upload real photos of figures to help with buying decisions.

- 🔁 **Trading Page**  
  List figures for trade or browse what others are offering/wanting.

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, SASS  
- **Backend:** Node.js / Express  
- **Database:** Store in a single json file  
- **Deployment:** Netlify (Frontend)
